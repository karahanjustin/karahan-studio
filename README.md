# Karahan Studio

Premium websites for US med-spas and aesthetic clinics. $5k fixed price, 14 days from kickoff to launch.

This repo contains:

- `index.html` — the agency landing page (positioning, work, pricing, FAQ)
- `work/glowmed/` — Demo 1: luxury Beverly Hills med-spa archetype
- `work/radiance/` — Demo 2: clinical, transparent-pricing PA-led practice
- `work/lumen/` — Demo 3: warm wellness studio with membership program
- `outreach/PLAYBOOK.md` — the full 90-day outreach playbook (read first)
- `outreach/templates.md` — copy/paste DM and email templates
- `outreach/loom-script.md` — the 90-second site audit recording structure
